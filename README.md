<h1 align="center">Netflix UI Clone</h1>

<p align="center">
A responsive front-end clone of the Netflix landing page built using 
<b>HTML</b> and <b>CSS</b>.  
This project recreates the visual design and layout of the official Netflix homepage.
</p>

<hr>

<h2>Overview</h2>

<p>
This project is a front-end recreation of the Netflix homepage interface. 
It was developed as part of learning responsive web development and modern UI layout techniques.
</p>

<p>
The website replicates the structure and styling of Netflix’s landing page including the hero section,
feature sections, FAQ area, and footer layout.
</p>

<h2>Page Sections</h2>

<h3>Navigation Bar</h3>

<p>
The top navigation bar contains the Netflix logo, language selector, 
and sign-in button similar to the official website.
</p>

<h3>Hero Section</h3>

<p>
The hero section introduces the platform with promotional messaging 
and a call-to-action email input field.
</p>

<h3>Feature Sections</h3>

<p>
Multiple sections highlight key Netflix features such as:
</p>

<ul>
<li>Streaming on TV</li>
<li>Downloading shows for offline viewing</li>
<li>Watching across multiple devices</li>
<li>Creating profiles for kids</li>
</ul>

<h3>FAQ Section</h3>

<p>
A Frequently Asked Questions section designed to match the layout 
commonly used on the Netflix website.
</p>

<h3>Footer</h3>

<p>
The footer contains multiple columns of links similar to the official Netflix page layout.
</p>
